Countdown365 is a flipclock countdown timer.

== Description ==

Countdown365 is a plug-in that allows you to quickly and easily add a flipclock and can be a game-changer for your woocommerce marketing strategy.

To use the plugin in your site, all you need to add to the page is the `[countdown365 enable_on_product_page=""  title="" date=""  language=""  face=""  label=""  background_color="" label_color=""  font_color="" separator_color=""  font_size="" border_radius="" digit_gap=""  dot_size=""  message="" ]` shortcode via gutenburg block.

= Attractive User Interface =
A good UI can sometimes makes a system even more attractive. countdown365 has an intuitive design that allows navigating easily. With it, you can manage your event and woocommerce marketing strategy.

= Shortcut / Hotkey Support =
countdown365 has shortcut key support that lets you use its features faster.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'countdown365'
3. Activate countdown365 from your Plugins page.


= From WordPress.org =

1. Download countdown365.
2. Upload the 'countdown365' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate countdown365 from your Plugins page.
